## Cómo cambiar / agregar

1. Guardá la foto en esta carpeta con el nombre indicado.
2. Si querés otro nombre, editá el campo `image` del producto en
   `src/data/products.ts` (ej: `image: "/products/mi-foto.jpg"`).

Recomendado: imágenes en proporción **4:3**, ~1200×900 px, fondo claro.
